# 🏗 Tworzenie aplikacji

```
npm install
npm run dev
```

Aby użyć [Nodemon'a](https://nodemon.io/), do automatycznego restartowania serwera po każdej zmianie w kodzie, należy
użyć polecenia:
```
npm run watch
```

# 💻 Wersja produkcyjna

```
npm install --production
npm run build
npm run start
```

# 🧪 Testy

```
npm run test
```
